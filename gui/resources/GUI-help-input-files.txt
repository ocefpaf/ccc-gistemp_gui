The "big" files...

SBBX.HadR2: Sea surface temperatures.

ftp://data.giss.nasa.gov/pub/gistemp/SBBX.HadR2

ushcnv2.gz: Temperature data from USHCN.

The location of this file varies (usually every month, as a
new release is made). Open,

ftp://ftp.ncdc.noaa.gov/pub/data/ushcn/v2/monthly

and choose the most recent file with the name
9641C_YYYYMM_F52.avg.gz (YYYY and MM are numbers
for year and month). Rename it ushcnv2.gz.

v2.mean: Uncompressed version of v2.mean.zip
normally created by ccc-gistemp automatically,
download v2.mean.zip instead.

v2.mean.zip: Temperature data from GHCN.

ftp://ftp.ncdc.noaa.gov/pub/data/ghcn/v2/zipd/v2.mean.zip
